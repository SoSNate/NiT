### 🔵 CORE — הגדרות ונוסחאות מרכזיות

*   **הגדרת פוטנציאל חשמלי (V):**
    *   "השדה החשמלי = הכוח החשמלי על יחידת מטען חיובי." (שיעור 2.pdf, עמוד 4)
    *   **מה חייבים לדעת בעל פה לבחינה:** פוטנציאל חשמלי הוא גודל סקלרי המתאר את האנרגיה הפוטנציאלית ליחידת מטען בכל נקודה במרחב. הוא קיים גם אם אין מטען בנקודה.

*   **קשר בין שדה חשמלי לפוטנציאל חשמלי (E = -∇V, ΔV = -∫E·dr):**
    *   הקשר בין השדה החשמלי (וקטור) לפוטנציאל החשמלי (סקלר) הוא שהשדה החשמלי הוא הגרדיאנט השלילי של הפוטנציאל. באופן אינטגרלי, השינוי בפוטנציאל בין שתי נקודות שווה למינוס האינטגרל של השדה החשמלי לאורך המסלול המחבר ביניהן.
    *   **מה חייבים לדעת בעל פה לבחינה:**
        *   `E = -∇V` (השדה הוא הגרדיאנט השלילי של הפוטנציאל)
        *   `ΔV = -∫E·dr` (הפרש פוטנציאלים הוא מינוס האינטגרל של השדה לאורך מסלול)

*   **פוטנציאל חשמלי של מטען נקודתי (V = kQ/r):**
    *   הפוטנציאל החשמלי במרחק `r` ממטען נקודתי `Q` נתון על ידי `V = kQ/r`, כאשר `k` הוא קבוע קולון.
    *   **מה חייבים לדעת בעל פה לבחינה:** הנוסחה הבסיסית לחישוב פוטנציאל של מטען נקודתי.

*   **עקרון הסופרפוזיציה לפוטנציאל חשמלי:**
    *   "השדה החשמלי בנקודה כלשהיא הוא הסכום הווקטורי של השדות החשמליים שיוצרים המטענים בסביבה." (שיעור 2.pdf, עמוד 6) - עקרון זה חל גם על הפוטנציאל החשמלי, אך מכיוון שהפוטנציאל הוא גודל סקלרי, הסכימה היא אלגברית (לא וקטורית).
    *   **מה חייבים לדעת בעל פה לבחינה:** הפוטנציאל הכולל במערכת הוא סכום הפוטנציאלים שכל מטען יוצר בנפרד.

*   **אנרגיה פוטנציאלית חשמלית של מערכת מטענים:**
    *   האנרגיה הפוטנציאלית של מערכת מטענים היא העבודה הדרושה כדי להרכיב את המערכת ממרחק אינסופי. עבור שני מטענים `q1` ו-`q2` המופרדים במרחק `r`, האנרגיה הפוטנציאלית היא `U = k * q1 * q2 / r`. עבור מערכת של `N` מטענים, יש לסכום את האנרגיה הפוטנציאלית של כל זוג מטענים.
    *   **מה חייבים לדעת בעל פה לבחינה:** הנוסחה לחישוב אנרגיה פוטנציאלית של זוג מטענים, והבנה שצריך לסכום על כל הזוגות במערכת.

### 🟡 PATTERN — 3 סוגי שאלות שחוזרות בבחינות HIT

1.  **שם:** חישוב פוטנציאל/שדה ממערכת מטענים נקודתיים.
    *   **מבנה טיפוסי:** נתונה מערכת של מספר מטענים נקודתיים במיקומים שונים. נדרש לחשב את הפוטנציאל הכולל או השדה החשמלי הכולל בנקודה מסוימת.
    *   **צעדי פתרון ממוספרים:**
        1.  זהה את כל המטענים הנקודתיים ואת מיקומם.
        2.  זהה את הנקודה בה נדרש לחשב את הפוטנציאל/שדה.
        3.  חשב את המרחק מכל מטען נקודתי לנקודה המבוקשת.
        4.  **לפוטנציאל:** חשב את הפוטנציאל שכל מטען יוצר בנקודה המבוקשת באמצעות `V = kQ/r`. סכום את כל הפוטנציאלים אלגברית (עקרון הסופרפוזיציה).
        5.  **לשדה:** חשב את השדה החשמלי שכל מטען יוצר בנקודה המבוקשת באמצעות `E = kQ/r²` בכיוון המתאים (ממטען חיובי החוצה, ממטען שלילי פנימה). פרק את וקטורי השדה לרכיבים (x, y, z). סכום את הרכיבים המתאימים של כל השדות הווקטוריים (עקרון הסופרפוזיציה).

2.  **שם:** חישוב עבודה או שינוי באנרגיה פוטנציאלית בעת הזזת מטען.
    *   **מבנה טיפוסי:** נתון מטען חשמלי במערכת עם מטענים נוספים. נדרש לחשב את העבודה שהכוח החשמלי מבצע או העבודה החיצונית הנדרשת כדי להזיז את המטען מנקודה א' לנקודה ב'.
    *   **צעדי פתרון ממוספרים:**
        1.  חשב את הפוטנציאל החשמלי הכולל `V_A` בנקודה ההתחלתית `A` (ללא המטען הנע).
        2.  חשב את הפוטנציאל החשמלי הכולל `V_B` בנקודה הסופית `B` (ללא המטען הנע).
        3.  האנרגיה הפוטנציאלית ההתחלתית היא `U_A = q * V_A`.
        4.  האנרגיה הפוטנציאלית הסופית היא `U_B = q * V_B`.
        5.  העבודה שהכוח החשמלי מבצע היא `W_electric = U_A - U_B = -q * (V_B - V_A)`.
        6.  העבודה החיצונית הנדרשת היא `W_external = U_B - U_A = q * (V_B - V_A)`.

3.  **שם:** קשר בין שדה לפוטנציאל במקרים של התפלגות מטען רציפה.
    *   **מבנה טיפוסי:** נתונה התפלגות מטען רציפה (מוט, טבעת, דיסקה). נדרש לחשב את השדה החשמלי או הפוטנציאל בנקודה מסוימת, או להשתמש בקשר `E = -∇V` כדי למצוא אחד מהם אם השני נתון.
    *   **צעדי פתרון ממוספרים:**
        1.  **אם נתון השדה ונדרש הפוטנציאל:** השתמש בנוסחה `ΔV = -∫E·dr`. בחר נקודת ייחוס (לרוב אינסוף, שם הפוטנציאל הוא אפס) וחשב את האינטגרל לאורך מסלול מהנקודה הזו לנקודה המבוקשת.
        2.  **אם נתון הפוטנציאל ונדרש השדה:** השתמש בנוסחה `E = -∇V`. חשב את הנגזרות החלקיות של הפוטנציאל לפי כל ציר (`Ex = -∂V/∂x`, `Ey = -∂V/∂y`, `Ez = -∂V/∂z`) ורכיב אותן לווקטור השדה.

### 🔴 TRAP — 2-3 טעויות נפוצות של סטודנטים שנה א'

1.  **טעות:** התייחסות לפוטנציאל כאל גודל וקטורי.
    *   **דוגמה שגויה:** חיבור וקטורי של פוטנציאלים ממטענים שונים.
    *   **תיקון והסבר:** פוטנציאל חשמלי הוא גודל **סקלרי**. לכן, כאשר מחשבים את הפוטנציאל הכולל בנקודה ממערכת מטענים, יש לסכום את הפוטנציאלים שכל מטען יוצר בנפרד **אלגברית**, תוך התחשבות בסימן המטען. `V_total = Σ V_i`.

2.  **טעות:** בלבול בין אנרגיה פוטנציאלית חשמלית לעבודה.
    *   **דוגמה שגויה:** חישוב העבודה כ- `q * ΔV` במקום `q * (V_final - V_initial)`.
    *   **תיקון והסבר:** העבודה שהכוח החשמלי מבצע כאשר מטען `q` עובר מנקודה `A` לנקודה `B` היא `W_electric = -q * (V_B - V_A)`. העבודה החיצונית הנדרשת כדי להזיז את המטען היא `W_external = q * (V_B - V_A)`. חשוב להבדיל בין העבודה שהמערכת מבצעת לבין העבודה שנדרשת להכניס למערכת.

3.  **טעות:** אי-התחשבות בסימן המטען בעת חישוב פוטנציאל.
    *   **דוגמה שגויה:** חישוב `V = kQ/r` עבור מטען שלילי `Q` והתייחסות ל-`V` כחיובי.
    *   **תיקון והסבר:** הנוסחה `V = kQ/r` כוללת את הסימן של המטען `Q`. מטען חיובי יוצר פוטנציאל חיובי, ומטען שלילי יוצר פוטנציאל שלילי. יש להקפיד להציב את סימן המטען בנוסחה.

### 🟢 EXAM — שאלה אחת מלאה מהבחינה שצוינה

**שאלה:**
נתונים שלושה מטענים נקודתיים: `q1 = +2 μC` ממוקם בראשית הצירים (0,0), `q2 = -3 μC` ממוקם בנקודה (0, 0.1m), ו-`q3 = +4 μC` ממוקם בנקודה (0.1m, 0).
א. חשב את הפוטנציאל החשמלי הכולל בנקודה P (0.1m, 0.1m).
ב. חשב את העבודה החיצונית הנדרשת כדי להביא מטען `q4 = +1 μC` ממרחק אינסוף לנקודה P.

**רמת קושי:** 2/5

**פתרון צעד-צעד מפורט:**

**חלק א': חישוב הפוטנציאל החשמלי הכולל בנקודה P (0.1m, 0.1m)**

1.  **זיהוי המטענים ומיקומם:**
    *   `q1 = +2 μC = +2 * 10^-6 C` ב- `(0,0)`
    *   `q2 = -3 μC = -3 * 10^-6 C` ב- `(0, 0.1m)`
    *   `q3 = +4 μC = +4 * 10^-6 C` ב- `(0.1m, 0)`
    *   נקודה P: `(0.1m, 0.1m)`
    *   קבוע קולון: `k = 9 * 10^9 Nm²/C²`

2.  **חישוב המרחקים מכל מטען לנקודה P:**
    *   מרחק מ-`q1` ל-P (`r1`):
        `r1 = sqrt((0.1m - 0m)² + (0.1m - 0m)²) = sqrt(0.1² + 0.1²) = sqrt(0.01 + 0.01) = sqrt(0.02) m`
        `r1 ≈ 0.1414 m`
    *   מרחק מ-`q2` ל-P (`r2`):
        `r2 = sqrt((0.1m - 0m)² + (0.1m - 0.1m)²) = sqrt(0.1² + 0²) = sqrt(0.01) = 0.1 m`
    *   מרחק מ-`q3` ל-P (`r3`):
        `r3 = sqrt((0.1m - 0.1m)² + (0.1m - 0m)²) = sqrt(0² + 0.1²) = sqrt(0.01) = 0.1 m`

3.  **חישוב הפוטנציאל שכל מטען יוצר בנקודה P:**
    *   `V1 = k * q1 / r1 = (9 * 10^9 Nm²/C²) * (2 * 10^-6 C) / (sqrt(0.02) m)`
        `V1 ≈ (18000) / 0.1414 V ≈ 127290 V`
    *   `V2 = k * q2 / r2 = (9 * 10^9 Nm²/C²) * (-3 * 10^-6 C) / (0.1 m)`
        `V2 = (9 * 10^9) * (-3 * 10^-6) / 0.1 = -27000 / 0.1 = -270000 V`
    *   `V3 = k * q3 / r3 = (9 * 10^9 Nm²/C²) * (4 * 10^-6 C) / (0.1 m)`
        `V3 = (9 * 10^9) * (4 * 10^-6) / 0.1 = 36000 / 0.1 = 360000 V`

4.  **חישוב הפוטנציאל הכולל בנקודה P (סכימה אלגברית):**
    *   `V_P = V1 + V2 + V3`
    *   `V_P ≈ 127290 V - 270000 V + 360000 V`
    *   `V_P ≈ 217290 V`

**חלק ב': חישוב העבודה החיצונית הנדרשת להבאת מטען q4 לנקודה P**

1.  **זיהוי המטען הנע והנקודות:**
    *   המטען הנע: `q4 = +1 μC = +1 * 10^-6 C`
    *   נקודת התחלה: מרחק אינסוף (שם הפוטנציאל הוא 0).
    *   נקודת סיום: נקודה P, שם הפוטנציאל הוא `V_P ≈ 217290 V`.

2.  **חישוב העבודה החיצונית:**
    *   העבודה החיצונית הנדרשת היא `W_external = q4 * (V_P - V_infinity)`.
    *   מכיוון ש-`V_infinity = 0`, הנוסחה היא `W_external = q4 * V_P`.
    *   `W_external = (1 * 10^-6 C) * (217290 V)`
    *   `W_external ≈ 0.217 J`

*   הבנה של הקשר `E = -∇V` ו-`ΔV = -∫E·dr` היא קריטית, אך דורשת הבנה של נגזרות חלקיות ואינטגרלים, מה שעשוי להיות מאתגר בשלב זה.
